# Betless Demo Script

Betless helps people protect money from gambling urges by turning savings into a commitment plan.

The user can create a vault by choosing either a one-time lock or a periodic top-up plan. In this demo, we’ll create a ₱10,000 savings goal with ₱2,000 monthly top-ups.

Once the vault is created, the user can see their progress, next top-up date, unlock schedule, and weekly reward eligibility.

The reward is not random and not game-based. It is a small positive reinforcement, such as a food or transport voucher.

For the Stellar workshop, this MVP uses Stellar testnet as the proof layer for the locked commitment concept. The app does not handle real GCash deposits, real vouchers, or real yield yet. Those are phase 2 through licensed partners.

Betless is not a gambling app. It is a commitment savings tool that helps users bet less, save more, and stay in control.
